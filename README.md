# Pixel RAG Assistant

基于 ChatGLM3 + BGE + FAISS + Gradio 构建的本地知识库问答系统。

## 功能

- 本地知识库问答
- PDF文件上传
- DOCX文件上传
- TXT文件上传
- 向量数据库FAISS
- ChatGLM3大模型回答
- 知识库动态更新
- 文件管理

## 技术栈

- ChatGLM3-6B
- BGE-small-zh-v1.5
- LangChain
- FAISS
- Gradio
